# 🎤 AI-Powered Interview Simulator

AI Interview Simulator is a web-based application that helps users practice technical interviews through real-time AI-generated questions, voice interaction, and automated evaluation.

---

## 🚀 Features

- 🧠 **AI Question Generation**  
  Generates topic-based conceptual interview questions

- 🎤 **Voice Input (Speech-to-Text)**  
  Answer questions using microphone

- 🔊 **AI Voice Output (Text-to-Speech)**  
  Questions are spoken like a real interviewer

- 📊 **Real-Time Evaluation**  
  Provides:
  - Score (1–10)
  - Feedback
  - Explanation

- 📈 **Performance Dashboard**
  - Total score  
  - Average score  
  - Question-wise performance  

- ⏱️ **Timer-Based Questions**  
  Simulates real interview pressure

- 🛑 **Manual Controls**  
  Start/Stop interview and mic control

---

## 🛠️ Tech Stack

- **Backend:** FastAPI  
- **Frontend:** HTML, CSS, JavaScript  
- **AI Model:** Groq (LLaMA 3)  
- **Charts:** Chart.js
